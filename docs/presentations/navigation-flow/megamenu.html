How the top nav actually opens.

Four mockups — one per top-level label — showing what drops down when a user hovers. Labels are not clickable; only the items inside the flyout link to pages. Anchor items sit at the top of each group, sized larger and color-tinted. /machines/ and the category landings stay loud and visible.

Conventions

Top-level label

Pure label, not a link. Hovering opens the flyout. Active state = colored underline.

Group header

Small mono caps. Names the user’s sub-job (See all machines, Help me choose, Compare).

Anchor item

Bigger, color-tinted, with the destination URL. The primary action in the group.

List items

Smaller, semibold. Each one is a real link to a real page; URL printed beside the name.

Key takeaway

Top labels don’t click. Flyouts have two equal lanes — browse and guided. /machines/ and every existing category and product page keep their URLs and stay visible. Configurator and Learning Center stay too; they move into the right action path.
