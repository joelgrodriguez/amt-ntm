## Recurring themes (read these first)

These show up on nearly every page — fix them as systemic theme rules, not one-offs:

1. **Heading line spacing is too loose.** Multiple pages: tighten multi-line heading `line-height` to match the page's top hero H1, which Evita repeatedly held up as the good example. ("Tighten up line spacing to be more similar to the H1", "Make the line spacing tighter like the heading at the top of the page", etc.) The About-page hero heading was called out as the *positive* reference: "this is an example of great line spacing for larger headings."
2. **Standalone headings/subheads should be centered + full-width.** When a heading/paragraph block sits on its own (not paired beside something), left-aligning it "just looks a bit empty." Applies to: About people section, About after-the-sale section, MACH II Family ("Three K-style configurations.", "Three MACH II machines. Pick yours."), gutter category Learning Center, SSQ3 "Rollforming Learning Center", roof/wall Learning Center.
3. **Hero text needs a black→transparent gradient overlay** for legibility over the photo. Requested on the combo, SSQ3, and roof/wall heroes — "a black overlay that fades out on the right," with a Honda Certified/Pre-Owned listing pasted in as the reference example. "Doesn't have to be as dark."
4. **Image backgrounds are too dark.** The dark navy product-image backgrounds should become "a very light tint of this color." The MACH II 5"/6" FEATURED card's light background is the explicit target ("Please make them look like this"). Hits combo, SSQ3, gutter category, MACH II Family.
5. **Use real in-field / transparent product photos, not stock or broken images.** Repeated: "Please use an image of the product in the field" / "Use a field shot if possible" / "Transparent SSQ3 product image here please" / a broken combo product image. She supplied assets (see Assets below).
6. **Color consistency.** Make section callout eyebrows + bullet points **blue, not red** ("only use red for buttons if we're doing 2 side by side" / "only use red if we have a secondary CTA"), and make all black/gray body text the **same** color across the site.
7. **Headline sections feel empty — add a short explainer paragraph** under the headline (or tighten the heading→content gap if no copy fits). Hits combo (profiles, accessories, full-details), MACH II Family comparison table, SSQ3 "Every Machine, Side by Side" and "Machines for Every Project".
8. **Image boxes look like they're floating — add a subtle stroke/shadow** to encapsulate them, and **make product images more uniform in size** across card grids.
